## 🌍 世界観

- Xは叩きが多すぎる
- インスタはキラキラしてないといけない
- Posiは「おめでとう」しか存在しない
- バーキン買った人も、宅建受かった人も、赤ちゃんが初めて立った人も、全員等しくおめでとうをもらえる
- **「今日早起きできた」「ジム行った」「禁煙3日目」レベルの小さな達成も等しくおめでとう**
- コーヒー飲みながらスワイプして、なんか自分も頑張ろってなれる場所
- 1000個おめでとうが集まったら花火が上がる（元気玉）
- **比較・ランキングは一切存在しない（思想の根幹）**

## 👤 ユーザー体験

### 見る側（メイン体験）
1. アプリ開く
2. 投稿がスワイプで流れてくる
3. ポチっ → おめでとう送信
4. 元気玉カウンター増える（今日のおめでとう 743/1000）
5. 1000個達成 → 🎆花火エフェクト

### 投稿する側
1. **達成したこと・嬉しいことを投稿（結果・事実のみ）**
2. 写真添付OK
3. カテゴリタグ選ぶ
4. 世界中からおめでとうが届く
5. **どの国からおめでとうが届いたか国旗で表示される**

### 通知
1. 好きなカテゴリを選ぶ（複数可）
2. **オンボーディング最初の1週間：1日5〜6個（中毒化）**
3. **通常時：1日2〜3個**
4. 通知からワンタップでおめでとう送れる

## 👤 プロフィール設計

### 「今挑戦してること」機能
- 任意入力（50文字以内）
- **開始日付を設定**
- **経過日数を自動計算・表示**

```
今挑戦してること
📅 2026年3月29日〜
「宅建取る」
━━━━━━━━━━
47日目
```

### 効果
- **自分がアプリを開く理由になる**（継続日数の可視化）
- **見る側の共感が深まる**（文脈として機能する）
- 「みんな頑張っとる」の循環が生まれる

### 投稿の定義（重要）
- ✅ **達成・結果**のみ投稿可能
- ❌ 意思・宣言・「頑張ります」は投稿不可
- → タイムラインの純度を保つ
- → 「まだやってないのにおめでとう」が発生しない

### プロフィール構成
```
プロフィール
├── 名前（匿名OK）
├── 今挑戦してること（任意・50文字）＋開始日付＋経過日数
└── 達成履歴（投稿一覧）
```

- 写真添付：OK

### Posiプレミアム（月100円）
- 投稿：無制限
- 通知：1日5個
- 投稿が優先的に通知配信される
- 花火エフェクトが豪華になる（おまけ）

## 🛡 モデレーション設計

### テキスト
- Claude API でファックワード・誹謗中傷を自動検出
- 投稿前にリアルタイムチェック → 弾いたらやさしくエラー表示

### 画像
- Google Cloud Vision API（SafeSearch Detection）
- グロ・成人向け・暴力 → 自動リジェクト
- 無料枠：月1,000回（初期は余裕）

### 通報
- 各投稿に通報ボタン設置
- さときが確認して即BAN

### 設計思想
- チャット機能なし（返信・コメント一切なし）
- リアクションは「おめでとう」のみ
- 批判・否定・マウントが構造的に不可能
- **ランキング・比較機能は一切なし**

## 🎆 元気玉システム

### 段階式カウンター（リセットなし・累計）

| マイルストーン | 演出 |
|---|---|
| 1,000個 | 🎆 花火 |
| 2,000個 | 🎆🎆 花火2倍 |
| 5,000個 | 🎇 超花火 |
| 10,000個 | 🎆×10 フィナーレ花火・以降花火機能終了 |

### ルール
- 累計カウント（毎日リセットなし）
- おめでとうポチのたびに +1
- マイルストーン達成 → アプリ開いている全員に花火エフェクト
- 10,000個達成でその動機は完結・花火機能は静かに引退
- カウンターは常時画面表示（例：「みんなのおめでとう 743 / 1,000」）
- **ランキングなし・全員で達成する感覚を維持**
